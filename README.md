What I'm building
Flowgate — Scenario-Driven Workflow Simulation Platform
Most testing tools mock APIs. Flowgate simulates real business workflows — loan approvals, payment flows, order fulfilment — step by step, with rule-based pass/fail logic, so QA teams can validate them without writing a single line of code.
Built with Python + FastAPI, React 19, PostgreSQL, and Keycloak. Clean Architecture. Multi-tenancy. Role-based access control. First time I'd touched Python when I started it.
It's unfinished. It's a personal project I build in downtime because the problem is real and the architecture is interesting. The README is honest about what works and what doesn't.
Go look at the code. That's more useful than anything I could write here.
