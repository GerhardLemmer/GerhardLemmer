## What I'm Building

### [Flowgate](https://github.com/GerhardLemmer/Testing-Platform) — Scenario-Driven Workflow Simulation Platform

Most testing tools mock APIs. Flowgate simulates real business workflows — loan approvals, payment flows, order fulfilment — step by step, with rule-based pass/fail logic. QA teams can validate entire workflows without writing code.

Stack: Python + FastAPI, React 19, PostgreSQL, Keycloak. Clean Architecture, multi-tenancy, role-based access control.

It's a personal project I built in my spare time because the problem is real. The README is honest about what's complete and what isn't. The code is the best way to evaluate it.
